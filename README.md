# Altus Search Website

A premium static recruitment website designed for deployment on GitHub Pages.

## Publish on GitHub Pages

1. Create a new public repository, for example `altus-search-website`.
2. Upload every file and folder from this package to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select `main` and `/ (root)`, then save.
6. Add the custom domain only after the preview is approved and DNS is ready.

## Before launch

- Replace the custom SVG wordmark with the official logo if required.
- Replace all sample jobs, placeholder statistics and sample insight articles.
- Connect forms to an approved secure backend, CRM or ATS.
- Add encrypted CV upload and documented retention/deletion controls.
- Obtain legal review of the privacy, cookie, terms and partnership pages.
- Verify the company name, address, licence statements, registration details and contact information.
- Add approved testimonials, team profiles, social links and recruitment licence information.
- Configure analytics only through a compliant consent workflow.
- Test accessibility, performance, browsers, forms and all mobile layouts.

## Files

- HTML pages are in the project root.
- Styles: `assets/css/styles.css`
- Interactions: `assets/js/main.js`
- Sample job data/filtering: `assets/js/jobs.js`
- Logo and graphics: `assets/img/`

No build step is required.
